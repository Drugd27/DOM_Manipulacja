# DOM_Manipulacja

Funkcje aplikacji:

Dodawanie produktu: Użytkownik może dodać nowy produkt na listę, klikając przycisk "Dodaj produkt" i wpisując nazwę w pole input.
Usuwanie ostatniego produktu: Klikając przycisk "Usuń ostatni produkt", ostatni element listy jest usuwany.
Edycja elementów: Kliknięcie na produkt umożliwia edycję jego nazwy, którą zapisuje się po naciśnięciu klawisza Enter.
Przyciski manipulacji: Użytkownik może dodać produkty na początek lub koniec listy, wyczyścić całą listę, a także przywrócić przykładowe elementy.
Kolorowanie elementów: Możliwość pokolorowania co drugiego elementu na liście.
Sortowanie: Produkty można posortować alfabetycznie.
Filtracja: Użytkownik może wpisać frazę w input, aby wyświetlić tylko produkty pasujące do wpisanej frazy.
Efekty wizualne: Dodanie efektu fadeOut i fadeIn przy edycji elementów.
Przeciąganie elementów: Użycie biblioteki jQuery UI do przeciągania i upuszczania elementów listy.

Użyte metody jQuery:

append(), prepend(), remove(), empty()
text(), html()
fadeOut(), fadeIn()
Manipulacja klasami: .addClass(), .removeClass(), .toggleClass()

Sortowanie i filtracja elementów.

Technologie:

HTML, CSS (opcjonalnie), JavaScript (jQuery)
jQuery UI (do obsługi przeciągania)
